# Aggregate requirements file – installs the Calliope framework + service layer.
# Used by the TEMPO installer (main.cjs) and manual dev setup.
#
# Subfiles:
#   requirements.calliope.txt  – Calliope energy modeling (Python 3.9–3.11)
#   requirements.pypsa.txt     – PyPSA energy modeling
#   requirements.adopt.txt     – ADOPT optimisation (experimental)
#   requirements.service.txt   – FastAPI / uvicorn service layer (always installed)
#
# Quick install (dev):
#   python -m venv .venv-calliope
#   .venv-calliope\Scripts\activate   (Windows)
#   pip install --prefer-binary -r python/requirements.txt
#
# Tested Python versions: 3.9, 3.10, 3.11
# Python 3.12+ is NOT supported by calliope 0.6.8.
